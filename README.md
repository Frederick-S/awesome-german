# awesome-german

* [Oxford](https://www.oxforddictionaries.com/us/translate/german-english/)
* [Collins](http://www.collinsdictionary.com/dictionary/german-english)

* [Wiktionary](https://de.wiktionary.org/wiki/Wiktionary:Hauptseite)
